GML Prompt ✖
Instructions for Constructing Nodes and Edges from the attached article Identify up to 40 key statements from the article or from quoted references, specifying the authors, year, and title. Categorize each statement based on its content, using a maximum of 15 categories. Ensure that all extracted information—including author, year, statement, title, statement, and category—do not contain any quotation marks, whether double ("), single ('), or curly (“ ” ‘ ’). Assign the cleaned and categorized information to the following node and edge attributes of a gml graph: Node Structure Each node must include the following attributes: id: A unique integer identifier for the node. type: Either "Actor" or "Concept". label: - For "Actor" nodes: the name of the author of the article. - For "Concept" nodes: the category label. description: - For "Actor" nodes: author, year, title from the references of the article or the article itself (source) - For "Concept" nodes: an empty string (""). Edge Structure Each edge must: Connect a source node of type "Actor" to a target node of type "Concept". Include the following attributes to the edge: source: The id of the "Actor" node. target: The id of the "Concept" node. statement: The actor’s statement (quotation from the article) referring to the concept, which reflects the assigned category. reference: the name of the file or source from which the statement was derived. agreement: Indicates the actor’s stance toward the concept: - "Yes" if the actor agrees with or supports the concept. - "No" if the actor opposes or rejects the concept. Connectivity Rules Every "Actor" node must be connected to at least one "Concept" node via an edge. There must be no isolated nodes; all nodes must be part of the network through at least one connection.
